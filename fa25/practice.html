Explanation bridge update

Practice answer explanations now use a numbers-first route: case fact → calculation bridge → journal or ledger logic → statement effect → self-check. For non-numeric framework questions, the same space uses a fact-to-classification bridge.

How to read the practice explanations

Use this once as the practice-section guide. Each model answer should then go straight to the case facts, number bridge, journal or ledger logic, statement effect, and self-check.

  • Inventory questions now distinguish perpetual stock-ledger workings, FIFO layers, moving weighted average, shortage and NRV write-downs.
  • Receivables questions now distinguish direct write-off, allowance adjustment, write-off under allowance, recovery reinstatement and aging schedules.
  • Cash or bank questions now distinguish bank-side timing items from book-side journal entries.
  • PPE or GST questions now separate carrying amount, proceeds excluding GST, output GST and gain or loss.
  • Equity or liability questions now flag redeemable preference shares and dividend or finance-cost classification.
