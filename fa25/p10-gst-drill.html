Explanation for GST journal-entry answer

GST split bridge

A Standard-rated sale:
$10,900 gross = $10,000 sales + $900 output GST

B Export sale:
$6,000 zero-rated = $6,000 sales + $0 GST

C Taxable purchase:
$2,180 gross = $2,000 purchase/expense + $180 input GST

Non-claimable / disposal bridge

D Non-claimable input tax:
$545 recorded fully as expense if input GST cannot be claimed.

E Asset disposal:
$3,270 gross = $3,000 net proceeds + $270 output GST

GST control bridge

Output GST:
$900 + $270 = $1,170

Less: input GST:
$180

Net GST payable:
$1,170 − $180 = $990

GST collected is a liability, not sales revenue. Recoverable input GST is an asset/reduction of GST payable, not an expense.
