Pewee Limited — GST receivable, payable and net GST accounts

GST control account answer bridge

AccountOfficial answer bridgeAmount
GST receivableInput GST after purchase returns and claimable expenses70,317 Dr then transferred
GST payableOutput GST after sales returns and taxable local outputs336,780 Cr then transferred
Net GSTPayable to tax authority266,463 Cr
Core self-check: net GST payable = output GST $336,780 - input GST $70,317 = $266,463.

Explanation bridge

Classification bridge: local standard-rated sales create output GST; local purchases from GST-registered suppliers create input GST; imports can create input GST; purchase returns and sales returns reverse the related GST.

Non-claimable and exempt bridge: salaries, CPF, some staff medical insurance, country club subscriptions and purchases from non-GST registered suppliers do not create claimable input GST. Rental of residential property and some exempt items are not treated as standard-rated output.
